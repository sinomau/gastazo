# Gastos Pareja 💑

App de control de gastos para parejas con Next.js + Supabase.

## Setup

### 1. Instalar dependencias
```bash
npm install
```

### 2. Variables de entorno
Creá un archivo `.env.local` con:
```
NEXT_PUBLIC_SUPABASE_URL=tu_url_de_supabase
NEXT_PUBLIC_SUPABASE_ANON_KEY=tu_anon_key
```

### 3. Correr localmente
```bash
npm run dev
```

### 4. Deploy
Conectá el repo a Vercel y configurá las variables de entorno en el dashboard.
